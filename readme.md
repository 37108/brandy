## Install
```
$ brew tap 37108/homebrew-tap
$ brew install aws-brandy
```

## Configuration
I strongly reccomend to create an alias like this.  
This alias automatically set environment.

```
alias brandy='`aws-brandy`'
```

## Usage
```
$ brandy
```
